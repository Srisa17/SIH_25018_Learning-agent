# SIH_25018_Learning-agent
SIH_25018_Learning-agent is multilingual learning agent was designed by the team "Enigma" of GITAM University for the SIH hackathon (Problem Statement: Telemedicine access to Nabha).

The purpose of this agent is to analyze patient symptoms—provided via multilingual voice or text input—and provide data-driven assistance to doctors during rural telemedicine consultations. Specifically, the agent is built to:
    1) Predict potential diseases based on the reported symptoms, keeping these predictions confidential and visible only to the consulting doctor.
    2) Suggest necessary diagnostic tests and recommend appropriate specialist referrals.
    3) Estimate the severity of the patient's condition and automatically trigger an SOS alert for critical cases.
    4) Continuously improve its accuracy over time through a built-in doctor feedback loop.
